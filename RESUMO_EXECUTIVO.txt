❓ DÚVIDAS COMUNS
─────────────────

P: Não aparece "Instalar app"?
R: Verificar:
   1. Manifest.json está linkado? <link rel="manifest">
   2. Service Worker registrou? (F12 → Application → SW)
   3. HTTPS ou localhost:3000 (não funciona em HTTP)

P: Offline não funciona?
R: Verificar:
   1. F12 → Application → Service Workers → Marcar "Offline"
   2. Recarregar → Deve servir cache
   3. Se não, limpar cache: Storage → Clear all

P: Funciona no desktop mas não no mobile?
R: Pode ser:
   1. Android: Chrome browser ✓ | Firefox ✗
   2. iOS: Safari 13+ ✓ | Chrome ✗ (usa Safari engine)
   3. Testar com URL HTTPS real (não localhost)

P: Ícone não aparece na tela inicial?
R: Verificar:
   1. icon-192.png e icon-512.png existem em /assets/img/
   2. URLs no manifest.json estão corretas
   3. Reinstalar app (limpar dados)
